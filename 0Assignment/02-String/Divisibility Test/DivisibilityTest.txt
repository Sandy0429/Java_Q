Divisibility Test
easy

Time Limit: 2 sec
Memory Limit: 128000 kB

Problem Statement
You are given a big number in form of a string A of characters from 0 to 9.

Input
The first argument is the string A.

Constraints
1 ≤ |A| ≤ 105

Output
Return "Yes" if it is divisible by 30 and "No" otherwise.

Example
Sample Input :
3033330

Sample Output:
Yes
