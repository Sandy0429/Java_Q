Count frequencies
easy
asked in interviews by 1 company

Time Limit: 2 sec
Memory Limit: 128000 kB

Problem Statement
Array List In Java

There is a list, having n integers that may have duplicates, is given. Write a Java program to print all unique elements and their frequencies. Elements must be in sorted order.

Input
There is an integer n is given in first line of input.
In Second line, n space separated integers are given.

Constraints
1 <= n <= 104

Output
Print all unique elements and their frequencies. Elements must be in sorted order.

Example
Sample Input:
7
1 2 4 3 5 4 3

Sample Output:
1 1
2 1
3 2
4 2
5 1
