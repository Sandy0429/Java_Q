Odd or Even

Time Limit: 2 sec
Memory Limit: 128000 kB

Problem Statement
Write a program to check whether a number is even or odd using switch case.

Input
First Line of the input contains the number n.

Constraints
1 <= n <= 1e9

Output
If the number is even print "Even" otherwise "Odd"

Example
Sample Input :
23

Sample Output :
Odd

Sample Input :
24

Sample Output :
Even
