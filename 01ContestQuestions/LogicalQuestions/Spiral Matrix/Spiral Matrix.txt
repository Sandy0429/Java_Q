Spiral Matrix
medium

Time Limit: 2 sec
Memory Limit: 128000 kB

Problem Statement
Given a NxM 2- D matrix. Print all elements of the matrix in clockwise spiral form starting from first row first column.

Input
First line contains n and m.
Next n lines contains m integers each.

Constraints
1 <= N, M <= 50
1 <= mat[i][j] <= 200

Output
output n*m integers separated by space.

Example
Input:
3 3
1 2 3
4 5 6
7 8 9

Output:
1 2 3 6 9 8 7 4 5

Explanation:
first row - > last column - > last row - > first column - > second row
