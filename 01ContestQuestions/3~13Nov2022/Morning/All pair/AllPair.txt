All pair
easy

Time Limit: 2 sec
Memory Limit: 128000 kB

Problem Statement
Given an integer N, find all possible pairs of A, and B such that A + B = N and A and B both are natural numbers.

Input
User Task:
Since this will be a functional problem, you don't have to take input. You just have to complete the function Allpair() that takes integer N as parameter.

Constraints:-
1 <= N <= 100000000

Output
Return the number of possible pairs of A and B.

Example
Sample Input:-
5

Sample Output:-
4

Explanation:-
(1, 4), (2, 3), (3, 2), (4, 1) are the only possible pairs.

Sample Input:-
2

Sample Output:-
1
